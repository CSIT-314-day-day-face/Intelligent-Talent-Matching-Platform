# Backend Module
This directory contains the server-side logic and matching algorithms for the platform.

## Key Responsibilities
- **Matching Logic:** Implementing the Top-K ($K=10$) recommendation system for candidates and Top-N ($N=10$) for employers.
- **API Development:** Handling requests between the frontend and the database.
- **Requirement Flexibility:** Logic is designed to be modular to accommodate the expected Week 8 requirement changes.
