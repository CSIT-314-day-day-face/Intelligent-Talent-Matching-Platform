# Frontend Module
This directory hosts the user interface and client-side assets.

## Components
- **Candidate Portal:** Forms for resume details (Education, Experience, etc.).
- **Employer Dashboard:** Interface for creating Job Descriptions (JD) and filtering candidates.
- **Search Interface:** Keyword-based search functionality for job postings.

## Design Tools
- **Figma:** Used for initial UI/UX mockups and wireframing.
